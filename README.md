# trouttricks.com

Static site for Trout Tricks, served from GitHub Pages.

## Regenerate Press Kit PDF

The "Download Press Kit (PDF)" button on [/media-kit.html](https://www.trouttricks.com/media-kit.html)
links to `/media/trouttricks-press-kit-2026.pdf`. That PDF is a print render of
the live media-kit page, generated by Playwright + headless Chromium.

Re-run after any edit to `media-kit.html`:

```sh
make press-kit
```

or directly:

```sh
python3 scripts/generate-press-kit-pdf.py
```

One-time setup:

```sh
pip3 install playwright
python3 -m playwright install chromium
```

The script fetches the **live** page from trouttricks.com, so push your
media-kit changes first, then regenerate the PDF, then commit
`/media/trouttricks-press-kit-2026.pdf`.
